# Contributing Guide

Спасибо, что хочешь улучшить проект!

## 🔧 Требования
- Node.js 20+
- NPM 10+
- TypeScript 5+
- Перед пушем запускай:
npm run lint
npm run test

## 📝 Коммиты
Мы используем **Conventional Commits**:
- feat: новая фича
- fix: исправление
- docs: документация
- test: тесты
- chore: обслуживание
- refactor: переработка без изменения логики

## 🔀 Ветки
- main — стабильная
- feature/* — новые фичи
- fix/* — багфиксы

## 🧪 Тесты
Все PR должны включать тесты или объяснение, почему тесты не нужны.

## 📦 Публикация релизов
Релизы создаются автоматически:
npm run release
