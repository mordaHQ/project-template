# Project Template

Официальная документация проекта.

## Возможности
- Автоматические релизы (standard-version + auto-release workflow)
- CI/CD (GitHub Actions)
- Форматирование кода (Prettier)
- Линтинг (ESLint)
- Тесты (Vitest)
- Защита main ветки
- Автоматические pull requests при релизах

## Структура проекта

- `src/` — исходный код
- `dist/` — сборка
- `.github/workflows/` — CI/CD
- `docs/` — документация (рендерится через GitHub Pages)
