# 🧰 Скрипты проекта

Проект использует набор npm-скриптов для запуска, тестирования, форматирования и релизов.

## ▶️ Запуск проекта

\\\bash
npm run dev
\\\

## 🧪 Тесты

\\\bash
npm test
npm run test:watch
npm run test:ui
npm run test:coverage
\\\

## 🧹 Качество кода

\\\bash
npm run lint
npm run format
\\\

## 📦 Сборка

\\\bash
npm run build
\\\

## 🚀 Релизы

\\\bash
npm run release
\\\
