## 🔄 Обновление документации

Документация:
- генерируется автоматически через **TypeDoc**
- обновляется при каждом деплое
- синхронизирована с актуальным кодом

> ⚠️ Не редактируй файлы внутри `api/*` вручную — они перезаписываются при генерации.
