# Project Template — документация

Кратко: это профессиональный TypeScript/NPM проект-шаблон с настроенным CI, тестами, Prettier/ESLint, Husky, lint-staged, GitHub Actions и автоматическими релизами.

## 🚀 Быстрый старт

```bash
# клонируем
git clone https://github.com/mordaHQ/project-template.git
cd project-template

# устанавливаем зависимости
npm ci

# запускаем тесты
npm test

# линтинг
npm run lint

# проверка форматирования
npx prettier --check .
